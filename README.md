# SakhaGames Frontend

Фронтенд для SakhaGames. Бэкенд находится в `../backend` и работает на Django REST Framework.

## Стек

- Nuxt 4
- Nuxt UI
- Tailwind CSS
- TypeScript

## Настройка

```bash
pnpm install
```

Переменная подключения к API:

```bash
API_URL=http://127.0.0.1:8000
```

## Разработка

```bash
pnpm dev
```

Проверка типов и линтинг:

```bash
pnpm typecheck
pnpm lint
```

## Сборка

```bash
pnpm build
```

```bash
pnpm preview
```
